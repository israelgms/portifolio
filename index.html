Sobre

Sobre

Olá. meu nome é Israel, tenho 22 anos e moro em Belo Horizonte, MG. Sou apaixonado por tecnologia, comecei a trabalhar bem cedo, com o foco em eletrônica e hardware. Um certo dia resolvi ir além, eu já conhecia bem sobre a parte que conseguimos tocar(hardware) mas sempre com uma curiosidade em desenvolvimento, foi ai que resolvi estudar como as coisas funcionam por baixo do pano. Por baixo do pano, uma expressão que se encaixaria a primeiro momento no backend, mas foi o Front-End que me conquistou. É a parte no qual todos tem aquele primeiro contato ao navegar na internet, e pra min foi um "amor a primeira vista". Toda a mágica acontece por aqui, e deve ser feito com muito carinho, atenção e conhecimento. O meu foco será sempre estudar o máximo possível e replicar o meu conhecimento, então fique a vontade para me acompanhar!

Conhecimentos

HTML

HTML5 (Hypertext Markup Language, versão 5) é uma linguagem de marcação para a World Wide Web e é uma tecnologia chave da Internet, originalmente proposto por Opera Software.[1] É a quinta versão da linguagem HTML.

CSS

CSS3 é a terceira mais nova versão das famosas Cascading Style Sheets (ou simplesmente CSS), pela qual se define estilos para um projeto web (página de internet).

JavaScript

JavaScript é uma linguagem de programação interpretada estruturada, de script em alto nível com tipagem dinâmica fraca e multiparadigma.

React JS

O React é uma biblioteca JavaScript de código aberto com foco em criar interfaces de usuário em páginas web. É mantido pelo Facebook, Instagram, outras empresas e uma comunidade de desenvolvedores individuais.

Bootstrap

Bootstrap é um framework web com código-fonte aberto para desenvolvimento de componentes de interface e front-end para sites e aplicações web usando HTML, CSS e JavaScript.

GitHub

é uma plataforma de hospedagem de código para controle de versão e colaboração. Ele permite que você e outras pessoas trabalhem juntos em projetos de qualquer lugar.
